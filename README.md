# R5.A.06_Rizzante--Madonna_Alexandre_Projet_Unity

Projet pour la ressource R5.A.06 auquel est utilisé Unity avec le package:
- Starter package

L'IDE utilisé est Visual Studio Code, le logiciel de modélsation 3d utilisé est Blender et le logiciel pour réaliser les textures est Gimp.

Tout ce qui concerne la partie Blender (compte rendu et fichiers se trouvent dans le dossier `/Rizzante--MadonnaBlender`.
Tout les objets Unity sont dans le dossier `/Assets/R5.A.06_Rizzante--Madonna_Alexandre_Projet_Unity` et le compte rendu Unity se trouve à la racine du projet (dépôt Git).
